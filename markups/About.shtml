About this site

This webpage was created in March 21st, 2010, the weekend after 2010 Spring Break, the second year of my PhD life.

The first version of this site was built inside the Kansas State University personal server back in 2009 fall semester. The idea was basically that I can have a place to exhibit myself online so some day when I am in the job market, people can easily see it and evaluate my teaching and research. Thanks to Dave Brown, a PhD graduate of Department of Economics, to share with me the template he used to set up his personal page. The template was drawn from the K-state system.

As a consequence, the CSS file of this site is attributed to Kansas State University. It is originally developed by James Asher, and modified by Jeff Croft in August 10th, 2004. This is the reason why you see KSU everywhere in my code. I hope this is not the abused use of the CSS file. I, however, have changed all colors and other minor place of the CSS file to make it fit my use.

Thanks for my friend Zhu Huang for helping me when I had some questions about deciding on a hosting company. I finally chose to try IX Web Hosting. It looks pretty cool to have a dedicated IP with this low price, about 113 USD for three years.
